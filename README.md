bootstrap-sass-demo
===================

botstrapp and sass

Steps:

1.- Yeoman
```
npm install -g yo
```

2.- Webapp
```
npm install generator-webapp
```

3.- Crear scaffold
```
mkdir mi-proyecto-chubibudubis
cd mi-proyecto-chubibudubis
```

4.- Crear el proyecto
```
yo webapp // elegir bootstrap-sass
```

5.- Dev server
```
grunt server
```

6.- Minificado y compilado
```
grunt build
// o
grunt
```
